# Visualizations

Charts and visualizations based on this dataset are available at:

**[unvow.com/divorce-statistics](https://unvow.com/divorce-statistics)**

Includes:
- State divorce rate bar charts
- International comparison charts
- National divorce rate trend line (2000–2023)
- Custody outcome segmented bars
- Child support payment and compliance charts
- Divorce cost breakdowns by state
- First divorce rates by race/ethnicity
- Remarriage rates by age and education

All visualizations are built with pure CSS/HTML for fast loading and accessibility.

---

Community-contributed visualizations are welcome. Submit a PR with your chart or notebook and we'll add it here.
