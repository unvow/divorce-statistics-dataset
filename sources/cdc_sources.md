# CDC National Vital Statistics System (NVSS)

## Overview

The CDC's National Vital Statistics System is the primary source for U.S. divorce and marriage rate data. NVSS collects data from state vital registration offices.

## Data Used

- **Divorce rates by state** — crude divorce rate per 1,000 population
- **Marriage rates by state** — crude marriage rate per 1,000 population

## Notes

- California, Georgia, Hawaii, Indiana, Louisiana, and Minnesota did not report divorce data to the CDC for several years. Recent estimates for these states use supplemental sources and should be treated as approximations.
- Rates are "crude" rates based on total population, not married population.
- Data year: 2023 (most recent available as of March 2026)
