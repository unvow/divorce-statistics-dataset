# U.S. Census Bureau Sources

## Current Population Survey — Child Support Supplement (CPS-CSS)

### Overview

The Census Bureau's CPS Child Support Supplement provides data on custodial parent demographics and custody arrangements.

### Data Used

- **Custodial parent breakdown** — 79.9% mothers, 20.1% fathers (P60-269)
- **Joint custody prevalence** — ~30.5% of noncustodial parents have some form of joint custody arrangement (P60-269, Table 4)
- **Child support compliance** — 69.3% receiving some payment, 43.5% receiving full amount (P60-269)

### Notes

- Report: "Custodial Mothers and Fathers and Their Child Support: 2017" (P60-269), published May 2020, using April 2018 CPS-CSS data
- "Custodial parent" = the parent with whom the child primarily resides
- The trend toward father custody has increased from 16% in 1994 to 20.1%
- Joint custody is measured separately from custodial parent designation

### Links

- Census Bureau CPS Reports: https://www.census.gov/topics/families/child-support.html
- Custodial Mothers and Fathers Report (P60-269): https://www.census.gov/library/publications.html

## Survey of Income and Program Participation (SIPP)

### Overview

The SIPP provides data on child support payment amounts received by custodial parents.

### Data Used

- **Average child support received** — $441/month ($5,292/year) among custodial parents who received payments
- **Total collected** — $20.2 billion

### Notes

- Report: "Child Support: 2021" published November 2023, using 2021 SIPP data
- State-level average payment data is NOT available from any single authoritative source
- OCSE does not publish average monthly payments by state

### Links

- Census Bureau SIPP: https://www.census.gov/programs-surveys/sipp.html
