# Contributing to the U.S. Divorce Statistics Dataset

Thank you for helping improve this dataset. Corrections and updates help journalists, researchers, and families access accurate information.

## How to Submit Corrections

1. **Open an Issue** — Describe the error, include the correct data, and cite a verifiable source (government report, court record, or official statistics).

2. **Submit a Pull Request** — Fork the repo, make your changes, and include:
   - The corrected data in the relevant CSV file
   - The source reference in the `source` column
   - An update to the corresponding file in `sources/` if it's a new source

## Data Standards

- All rates are per 1,000 population unless otherwise noted
- Dollar amounts are in USD
- Include the `year` and `source` columns for every row
- Use state abbreviations from USPS standard (e.g., CA, TX, NY)

## What We Accept

- Corrections to existing data with verifiable sources
- New state-level data from official court or government reports
- Updated figures from newer editions of existing sources

## What We Don't Accept

- Data without a verifiable government or institutional source
- Estimates from unverified surveys or opinion polls
- Promotional content or links to commercial services
